# logseq-case-converter-plugin
A quick utility to make the contents of a block into, upper case, lower case, title case or sentence case. 
![Screen Recording 2022-02-14 at 4 54 33 PM](https://user-images.githubusercontent.com/80150109/153868682-490904c4-20c6-4f64-9f85-7b65ee8f98e2.gif)

## Usage
1. Right click the bullet point of the block you want to change the case of
2. Search the menu for whichever operation you want to perform.
